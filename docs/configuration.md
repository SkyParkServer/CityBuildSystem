# Konfiguration

## `config.yml`

Wichtige globale Einstellungen:

- Prefix und Standardnachrichten
- Datenbankverbindung
- Servername (`server-name`) fuer servergebundene Features

## `homes.yml`

Definiert:

- Home-Limits je Rang
- Kaufpreise / Preissteigerung
- Gesamtlimit kaufbarer Homes

## `spawn.yml`

Definiert:

- Spawn-Location
- Ob bei Join automatisch zum Spawn teleportiert wird
