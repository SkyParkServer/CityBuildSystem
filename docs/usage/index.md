# Usage Ueberblick

Dieser Bereich richtet sich an Server-Betreiber und Teammitglieder, die das Plugin nutzen und konfigurieren.

## Inhalt

- Voraussetzungen und Setup fuer den ersten Start
- Konfiguration der YAML-Dateien
- Command-Referenz mit Permission-Hinweisen
- Feature-Ueberblick und wichtige Verhaltensregeln

## Ziel

Nach dem Lesen dieses Bereichs solltest du in der Lage sein, den Server korrekt zu starten, das Plugin einzurichten und die wichtigsten Funktionen sicher zu bedienen.
