# Commands

Kurze Referenz der wichtigsten Commands.

## Utility

- `/wb`, `/workbench`, `/craft`
- `/anvil`
- `/repair`
- `/wetter <regen|gewitter|sonne>`
- `/hat`
- `/tresor`
- `/v`, `/vanish`

## Homes / Teleport

- `/setspawn`, `/spawn`
- `/sethome <name>`, `/home <name>`, `/delhome <name>`, `/homes`
- `/setwarp <name>`, `/warp <name>`, `/warps`, `/delwarp <name>`

## Kommunikation

- `/msg <spieler> <nachricht>`
- `/r <nachricht>`
- `/msgtoggle`
- `/werbung <nachricht>`

## Economy

- `/balance`, `/money`
- `/eco ...`
- `/bank ...`
- `/kristalle ...`

## Moderation / Admin

- `/invsee <spieler>`
- `/near`
- `/cb ...`

> Die finalen Permissions stehen in `src/main/resources/plugin.yml`.
