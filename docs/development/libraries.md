# Libraries und Integrationen

## Erlaubte Abhaengigkeiten

- Paper API
- MySQL Connector/J (runtime via `plugin.yml` libraries)
- Lombok (compile-time)

## Integrationen

- Vault
- LuckPerms
- PlotSquared v7
- FastAsyncWorldEdit
- Multiverse-Core

## Nicht erlaubt

- Zusätzliche Runtime-Libraries ohne explizite Projektentscheidung
- Frameworks, die die Core-Regeln (DB/Cache/Message-Format) umgehen
